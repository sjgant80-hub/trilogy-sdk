trilogy-sdk
GitHub npm Framework
v1.0 — zero dependencies

One install.
Three AI APIs.

DocMind for documents. Deep for research. Flux for content. Full TypeScript. Retries built in. Ship in 5 minutes.

$ npm install trilogy-sdk
Input

DocMind

Documents in, structured data out. Receipts, invoices, bank statements, contracts.

doc.parse.receipt(file)
doc.parse.invoice(text)
doc.parse.bankStatement(pdf)
doc.parse.contract(buffer)
doc.chat(messages)
Intelligence

Deep

Questions in, research reports out. Citations, fact-checking, entity comparison.

deep.research(query)
deep.quick(query)
deep.factCheck(claims)
deep.compare(entities)
deep.extract(url)
Output

Flux

Briefs in, content out. Blogs, threads, emails, ads, landing pages, SEO.

flux.generate.blog(brief)
flux.generate.thread(brief)
flux.generate.email(brief)
flux.repurpose(content)
flux.brandVoice(samples)
